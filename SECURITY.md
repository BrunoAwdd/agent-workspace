# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| 0.1.x   | :x:                |

## Reporting a Vulnerability

Security is a high priority for the multi-agent orchestration layer, as compromised API access could allow rogue actors to poison tasks or manipulate sensitive operational data.

Please **do not** open public issues for security vulnerabilities. Instead, send an email directly to the project maintainers with a Proof of Concept (PoC). We will assess the bug privately and, if applicable, push out a patch responsibly before public disclosure.
